## Deployment

1. Clone the repo `git clone https://github.com/mimipeshy/expense-tracker-react.git`
2. run `npm run i` to install dependencies
3. run `npm start` to start the server
4. Open `http://localhost:3000/`
5. Enjoy :)

## Testing

1. Run `npm test`
